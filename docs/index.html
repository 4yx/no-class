No Class

A CSS Framework Without Class

No CSS selectors were used in the making of this HTML document*

No Class is a small CSS framework indented to make you write HTML that looks good, cross-browser, without having to pull in a massive CSS library like Bootstrap or Foundation and mock about with the custom markup and classes they bring. Don't get me wrong, they do a great job, but sometimes you just want to write plain HTML to “strut your stuff” without having it look like it was designed in 1999.

How to Install

Method 1: Install on your own server

  1. Download the latest release from here
  2. Transfer the CSS files to your website using FTP or via File Manager
  3. Link to them within your HTML file

Method 2: Use our ready-made CSS files

  1. Link to our ready-made CSS files using:

    Beautified: 
     <link rel="stylesheet" href="https://www.noclass.tk/css/no-class.css">
    Minified: 
     <link rel="stylesheet" href="https://www.noclass.tk/css/no-class.min.css">

If you followed these steps, you would have successfully implemented No Class to your website!
